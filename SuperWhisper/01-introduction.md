# SuperWhisper - Introduction

## Vue d'ensemble
SuperWhisper est une solution de dictée basée sur l'IA qui transforme instantanément la parole en texte. L'application est disponible sur **macOS**, **Windows** et **iOS**.

## Fonctionnement en 3 étapes
1. **Dictation** : L'utilisateur parle naturellement
2. **Transcription** : L'IA convertit la parole en texte avec précision
3. **Traitement intelligent** : L'IA adapte le texte aux besoins spécifiques (optionnel)

## Plateformes
- **macOS** : Application desktop complète
- **Windows** : Application desktop
- **iOS** : iPhone et iPad

## Modèle économique
- Version gratuite : 15 minutes de fonctionnalités premium
- Version Pro : Abonnement avec accès illimité
