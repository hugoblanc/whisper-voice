# SuperWhisper - Paramètres

## Accès aux paramètres
1. **Barre de menu** : clic sur logo → Settings
2. **Raccourci** : ⌘ + , (si "Show in Dock" activé)
3. **Mini fenêtre** : clic droit → Open Settings
4. **Deep Link** : `superwhisper://settings`

## Raccourcis Clavier

### Toggle Recording
- Démarre/arrête l'enregistrement
- Touches modificatrices seules possibles (Cmd, Fn)

### Cancel Recording
- Annule immédiatement (< 30 sec)
- Confirmation pour enregistrements plus longs

### Switch Mode
- Bascule entre modes pendant la dictée
- Maintenir modificateur + touche assignée

### Push-to-Talk
- Appuyer et maintenir pour enregistrer
- Relâcher pour arrêter
- Peut partager le même raccourci que Toggle

### Mouse Shortcut
- Clic rapide = toggle enregistrement
- Appui maintenu = push-to-talk

## Paramètres Audio

### Périphérique d'entrée
- Sélection manuelle du microphone
- Ou sélection automatique (défaut système)

### Ajustement du volume
- Automatique recommandé
- Capture voix à niveau optimal

### Suppression du silence
- Améliore les transcriptions
- Prévient les hallucinations IA

### Effets sonores
- Signal de début de dictation
- Son de fin de traitement
- Volume indépendant ou désactivé

## Paramètres Avancés

### Fenêtre d'enregistrement
- Désactivation possible
- Mini fenêtre minimale
- Fermeture auto après dictation

### Application
- Affichage dans le Dock macOS
- Enregistrement via clic icône menu

### Traitement vocal
- Durée active du modèle (10 sec → 1 heure)
- Normalisation dynamique du volume

### Contrôles de saisie
- Coller automatiquement les résultats
- Restaurer presse-papiers antérieur
- Shift pour envoyer auto
- Simulation de frappes (apps avec restrictions)

### Synchronisation
- Fichiers config dans ~/Documents/superwhisper
- Sync entre installations via licence
