Voice-controlled Cursor for any text field. Highlight an email, speak *"make it shorter and less formal"*, and the edit is pasted back in place.

Features

Built for the way you actually write.

6 processing modes

Brut, Clean, Formel, Casual, Markdown, Super. Shift-cycle during recording. Or create your own with custom prompts.

Auto-mode by app

Map Slack → Slack mode, Mail → Formel. Whisper Voice detects the frontmost app and switches automatically. Resets to Brut between unrelated apps.

Projects with smart tagging

History auto-tags entries to your projects via git remote, workspace hint, browser host. Filter, search, retro-tag in bulk.

Full pipeline audit

Every dictation logs input, system prompt, output, char delta. Built-in log viewer. You always know exactly what the LLM did with your voice.

Dual-format clipboard

HTML + plain text. Slack, Notion, Gmail get rendered bold/code/lists. Terminal and editors get the raw markdown. No <br> nonsense.

Custom vocabulary

Add your product names (PostHog, Chatwoot…) once. Biases Whisper AND the LLM prompt so your jargon survives transcription and reformatting.

Why I built this

Paid €10/month for what a weekend could ship.

I was paying €10/month to Superwhisper. Great app, but I used it 5 minutes a day and I knew the tech was an OpenAI Whisper API call behind a UI.

So I ran the numbers: €120/year × thousands of users = a fat margin on top of a ~$0.06/hour API cost. I figured the UI was the only real moat.

Three weekends later, Whisper Voice had feature parity + a few things Superwhisper doesn't have (Super Mode, project auto-tagging, dual-clipboard). I open-sourced it under MIT.

If you dictate, I want this to be the last voice-to-text tool you ever install. If you want to fork, rebrand, or ship your own flavor — go for it.
